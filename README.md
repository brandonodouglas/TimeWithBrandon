# ⏰ TimeWithBrandon

## 👀 What is it?
A productivity app designed specifically 
for people with ADHD, Autism and mental 
health needs

## 🔍 Problem it solves
Many productivity apps assume neurotypical 
users — this app adapts to how 
neurodivergent people actually think with a key focus on routines, time tracking, progress bars and the like. This helps people with executive functioning needs - ensuring they can maximise productivity without burning out.

## 💪 Features
- Routine timer with visual cues
- Simple distraction free interface
- Gentle reminders 
- Hyperfocus timeblocking

## 💻 Built with
React Native
